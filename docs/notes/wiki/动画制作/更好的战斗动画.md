`2.6.0` 版本对更好的战斗添加了进一步的联动，增加了一个变量用于检测当前的攻击动作。

可通过变量配合动画控制器，实现更精细的 Better Combat 动画。

## ctrl.bcombat_attack_animation

该变量提供`更好的战斗`当前播放的动画，下表仅列出原版`更好的战斗`模组的动画

|             Molang值              |      动作描述      |
| :-------------------------------: | :----------------: |
| one_handed_slash_horizontal_right | 单手武器由右向左劈 |
| one_handed_slash_horizontal_left  | 单手武器由左向右劈 |
|          one_handed_stab          |    单手武器刺击    |

查询更多动画请使用`/ysmclient watch var ctrl.bcombat_attack_animation`
